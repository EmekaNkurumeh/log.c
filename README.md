# log.c
a tiny logging module for c based on log.lua
## installation
```c
#include "log.h"
```

## usage
* **TRACE(...)**
* **DEBUG(...)**
* **INFO(...)**
* **WARN(...)**
* **ERROR(...)**
* **FATAL(...)**
